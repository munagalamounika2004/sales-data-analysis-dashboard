# Sales Data Analysis Dashboard
Project Overview  :
This project analyzes sales data to identify trends, regional performance, and top-performing products.

 Tools Used :
 Excel (Data Cleaning)
 Power BI (Dashboard)
 
 Key Insights :
* West region contributes highest sales
* Sales peak during year-end months
* Technology category dominates revenue
* High-value transactions generate majority of sales
